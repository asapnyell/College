Painel Interativo do Aluno 🎓

Este projeto é uma aplicação React desenvolvida para a atividade prática de criação de componentes, gerenciamento de estado e comunicação entre componentes (Props e State).

👥 Integrantes
Danyel Henrique

Gabriel Damasceno

🚀 Tecnologias Utilizadas
React.js (Biblioteca principal)

Vite (Build tool)

CSS3 (Estilização Global e Modular por Componente)

JavaScript (ES6+)

🛠️ Funcionalidades
Componente Perfil: Exibição dinâmica de informações do aluno via props.

Componente Ação: Botão que utiliza comunicação indireta (Filho -> Pai) para disparar eventos.

Estado Central: Uso de useState no componente App.jsx para atualizar o status do aluno em tempo real.

Estilização Híbrida: Separação entre estilos globais (App.css) e estilos específicos (Perfil.css e Acao.css).

📦 Como rodar o projeto
Caso queira testar a aplicação localmente, siga os passos abaixo:

Extraia o conteúdo do arquivo .zip.

Abra o terminal na pasta raiz do projeto.

Instale as dependências:

Bash
npm install

Inicie o servidor de desenvolvimento:

Bash
npm run dev

Acesse o link gerado no terminal (geralmente http://localhost:5173).
